Position: Static (Default)

Static Box

Static: Ignores top/right/bottom/left, stays in normal flow.

Position: Relative

Relative Box

Relative: Stays in flow but offset visually; space in flow is reserved.

Position: Absolute

Absolute Box

Absolute: Removed from flow, positioned relative to nearest positioned ancestor (container).

Position: Fixed

Fixed Box

Fixed: Anchored to viewport, stays in place on scroll.

Position: Sticky

Sticky Box

Sticky: Toggles between relative and fixed based on scroll within container.
